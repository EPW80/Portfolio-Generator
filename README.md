# Portfolio-Generator

## Purpose
The purpose of this application is to test out the capabilities of node.js. This portfolio generator was completed by using the
terminal only using node.js. The completed project itself will not deploy to GitHub pages so you will need to clone the repository
to your desktop in order to see the final results of the project.

## Built With

- JavaScript
- node.js
